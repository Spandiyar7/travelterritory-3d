# 3D-модели (GLB)

Сейчас все 3D-объекты сцены **процедурные** — собраны кодом в
`src/components/three/` и не требуют файлов моделей. Они полностью рабочие и
выглядят премиально без внешних ассетов.

## Если появятся реальные GLB-модели

Положите файлы сюда:

```
globe.glb              — планета/глобус
airplane.glb           — самолёт
suitcase.glb           — чемодан (опционально)
passport.glb           — паспорт / посадочный талон
hotel-card.glb         — карточка отеля (опционально)
```

## Как заменить процедурную модель на GLB

В нужном компоненте (например `AirplaneModel.tsx`) подключите загрузчик drei:

```tsx
import { useGLTF } from "@react-three/drei";

export function AirplaneModel() {
  const { scene } = useGLTF("/models/airplane.glb");
  return <primitive object={scene} />;
}
// useGLTF.preload("/models/airplane.glb");
```

Процедурную геометрию из компонента при этом можно удалить или оставить как
фолбэк. Компоненты сцены изолированы — замена одной модели не затрагивает
остальные.

Не используйте тяжёлые модели: для фонового слоя достаточно лёгких GLB
(до ~1–2 МБ), иначе пострадает производительность.
