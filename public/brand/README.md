# Брендовые ассеты

Папка для логотипов и фирменных изображений Travel Territory.

Рекомендуемые файлы:

```
logo.svg            — основной логотип (если появится векторный)
og-image.jpg        — изображение для соцсетей, 1200×630 px
favicon.ico         — уже лежит в src/app/favicon.ico
```

Сейчас логотип в шапке/подвале собран кодом (иконка + текст). OG-изображение
можно подключить, добавив `opengraph-image.jpg` в `src/app/` — Next.js
автоматически использует его для метатегов Open Graph.
