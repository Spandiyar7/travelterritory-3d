# Фото команды

Фотографии сотрудников/офиса для страницы «О компании».

Имена файлов произвольные (например `manager-1.jpg`, `office.jpg`,
рекомендуется 1200×1500 px). После добавления используйте компонент
`PassportPhotoFrame` (`src/components/frames/PassportPhotoFrame.tsx`),
передав путь в проп `src`, и установите `SHOW_PHOTOS = true` в
`src/config/site.ts`.

Без фото показывается премиальная рамка-плейсхолдер.
