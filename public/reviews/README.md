# Фото для отзывов

Сейчас в отзывах используются аватары-заглушки с инициалами — фото не требуются.

Если подключаете реальные отзывы с фотографиями авторов, кладите изображения
сюда и расширьте тип `Review` в `src/types/review.ts` полем `avatar`, затем
используйте его в `src/components/frames/ReviewImageFrame.tsx`.

Демо-отзывы заданы в `src/data/reviews.ts` и помечены как демо. Рекомендуется
заменить их виджетом реальных отзывов 2ГИС / Google / Яндекс.
