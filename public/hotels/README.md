# Фото отелей

Фотографии отелей для блока «Рекомендуемые отели» на страницах направлений.

Имена файлов произвольные (например `rixos-side.jpg`). После добавления файла
передайте его путь в проп `image` компонента `HotelImageFrame`
(`src/components/frames/HotelImageFrame.tsx`) и установите `SHOW_PHOTOS = true`
в `src/config/site.ts`.

Без фото показывается премиальный плейсхолдер с названием отеля.
