# Фото горящих туров

Фотографии отелей/курортов для карточек горящих туров.

## Имена файлов (точно)

```
turkey-side.jpg          uae-dubai.jpg
thailand-phuket.jpg      egypt-sharm.jpg
vietnam-nha-trang.jpg    maldives-resort.jpg
georgia-batumi.jpg       sri-lanka-bentota.jpg
```

## Как подключить

1. Положите файлы с этими именами сюда (рекомендуется 1600×1100 px).
2. В `src/config/site.ts` установите `SHOW_PHOTOS = true`.

Пути и сами туры (демо-данные) — в `src/data/hotTours.ts`.
Замените демо-туры на реальные предложения из CRM / API туроператора.
